## Run Locally

Clone the project
```bash
  git clone https://github.com/maidastach/swapi
```

Go to the project directory
```bash
  cd swapi
```

Install dependencies
```bash
  npm install
```

Start the React server
```bash
  npm run dev
```

Run the application at
```bash
  http://localhost:3000
```
